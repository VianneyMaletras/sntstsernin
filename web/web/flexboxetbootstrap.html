Introduction Le modèle Flexbox (ou boîte flexible) est une méthode de disposition en CSS qui permet de créer des interfaces adaptatives de manière simple et efficace. Il est particulièrement utile pour organiser des éléments sur une page, que ce soit en ligne (horizontalement) ou en colonne (verticalement).

                    
                      section {
                        display: flex;
                        border: 3px solid black;
                      }

Bootstrap est un framework CSS open-source qui facilite la création de mises en page réactives. Il est basé sur Flexbox et propose des classes CSS prédéfinies pour gérer rapidement l'alignement et la disposition des éléments.
