Tim Berner Lee

Il est considéré comme le père du web.
En 1989, il a rassemblé plusieurs idées et technologies pour construire un système facilitant l’échange de ressources entre les scientifiques du CERN.
Quand le web est tombé dans le domaine publique, le développement a été très rapide.

Il faut écrire un fichier html. Cette page possède toujours le même squelette.

La partie entre les balises head sert à discuter avec le navigateur

La partie entre les balises body sert à mettre tout ce que l'on souhaite afficher aux visiteurs

Pour que le navigateur sache ce qu'il doit afficher on met des balises. 

                      
                        <img src="IMG/lee.jpg" alt="Tim berner Leee" >
    Par exemple pour ajouter une image :
  • src="IMG/lee.jpg" : Identifie le chemin et le nom du fichier image. Peut être une adresse externe
  • alt="Tim berner Leee": description textuelle de l'image.
